````md
# 🎬 Intro Changer

## 📌 Overview

A PySide-based tool to replace intro sections of videos dynamically.

### 🚀 Features

✅ Cuts out old intros and inserts new ones  
✅ Supports multiple video formats  
✅ Fast processing

### 🔧 Installation

```sh
pip install -r requirements.txt
python main.py
```
````
